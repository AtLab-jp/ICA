Sound Source Separation with ICA
=================================

## What is ICA?
ICA means "Independent Computing Architecture".

## Requirement
- OS : MacOS Catallina 10.15.2
- Python 3.7.4
- TensorFlow 2.0.0

## Installation
```sh
$ git clone https://github.com/AtLab-jp/ICA
$ cd ICA/
$ pip install -r ./lib/requirement.txt
```
